# nine8 - Premium Digital Payment Platform

A premium fintech platform designed for Indian universities. Pay tuition fees, manage finances, and track spending seamlessly.

## Features

- **Secure Payments:** State-of-the-art encryption for safe transactions.
- **Tuition Management:** Easily handle fee payments and schedules.
- **Finances Tracker:** Monitor your spending and upcoming payments. 
- **Modern UI:** Built with enterprise-grade React, Tailwind, and Shadcn UI components.

## Getting Started

To run this project locally:

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`

Navigate to `http://localhost:8080/` to view the application.
